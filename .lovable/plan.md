

# Asociación de Baile Flamenco — Web One-Page

## Visión
Web single-page premium y editorial que transmite pasión, elegancia y modernidad. Paleta flamenca (rojo burdeos, negro, marfil, dorado), tipografía serif+sans-serif, animaciones suaves al scroll, alto contraste entre secciones.

## Estructura y Secciones

### 1. Hero (Portada)
- Fondo de imagen a pantalla completa con overlay oscuro y sutil patrón de lunares
- Titular emocional grande en serif ("Donde el arte se hace movimiento")
- Subtítulo breve + botón CTA "Únete" en rojo burdeos
- Navegación fija superior minimalista con scroll suave a secciones

### 2. Sobre Nosotros
- Fondo marfil/beige cálido
- Layout editorial: texto a un lado, imagen decorativa al otro
- Breve historia, misión y valores (cultura, comunidad, pasión)
- Detalles dorados sutiles como separadores

### 3. Clases / Actividades
- Fondo oscuro (negro) con texto claro
- 3 tarjetas elegantes: Flamenco, Sevillanas, Palillos
- Cada una con icono/ilustración, título y descripción breve
- Efecto hover sutil con escala y borde dorado

### 4. Galería
- Grid visual responsive (masonry-style) con imágenes placeholder
- Efecto hover con overlay y texto
- Fondo claro

### 5. Horarios / Ubicación
- Tabla de horarios estilizada con la paleta flamenca
- Sección de ubicación con dirección e info de contacto
- Fondo oscuro

### 6. Comunidad / Impacto
- Fondo beige con estadísticas destacadas (años, alumnos, actuaciones)
- Texto sobre el carácter sin ánimo de lucro, inclusión y comunidad
- Elementos decorativos sutiles

### 7. Contacto / CTA Final
- Formulario de contacto elegante (nombre, email, mensaje)
- Links a redes sociales
- CTA final emocional
- Footer con info legal mínima

## Diseño y UX
- Tipografía: Playfair Display (serif, títulos) + Inter (sans-serif, cuerpo)
- Animaciones fade-in al scroll con Intersection Observer
- Alternancia de fondos claros/oscuros entre secciones
- Patrón sutil de lunares como textura decorativa via CSS
- Navegación sticky con scroll suave
- Totalmente responsive (mobile-first)
- Idioma: Español (España)
